# html的标签

## 1. 分类
1. 单标签，如`<img />`，`<input />`
2. 双标签，如`<a></a>`, `<div></div>`

## 2. 标签间的关系
4. 并列关系
```html
<marquee>good day!</marquee>
<input />
```
<marquee>good day!</marquee>
<input />

2. 嵌套关系
```html
<marquee>good day! <input /></marquee>
```
<marquee>good day!    <input /></marquee>
